Avaliação V

Uso do robô Pioneer - pt. 2

Arquivo do Projeto

Prototipagem

Nesta simulação, já se entrega um cenário mais complexo, com obstáculos e curvas, para o pequeno robô Pioneer. Pra este, será necessária uma codificação mais complexa, caso contrário o robô ficará completamente desordenado.



Codificação

Mais uma vez iniciamos o código em Lua com a inicialização dos elementos: motores, sensores e variáveis. Então inicia a programação das ações do pequeno Pioneer: o primeiro passo é criar algumas variáveis para salvarem os elementos buscados pelos sensores. Depois de lidos os dados, entramos nas primeiras condições, estas estabelecem que, se os dados lidos foram nulos (nil), serão substituídos pelo valor 1. Definidos os valores de entrada, entramos na segunda condicional, que definem se o Pioneer seguirá em frente, dobrará para a direita ou para a esquerda para evitar bater em obstáculos.
